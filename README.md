AT Door - Service Booking & Professional Marketplace📌 Project OverviewAT Door is a robust Full-Stack mobile application designed to connect clients with professional workers. It streamlines the process of finding, booking, and communicating with service providers through an organized digital workflow.  
🔄 Key Workflow (The Business Logic)Search & Booking: Clients can search for professionals and request appointments.  Order Management: Workers (Employees) receive requests and can send personalized price quotes.Acceptance/Rejection: Clients have the full control to accept or reject the worker's offer.Feedback System: A rating and commenting system is implemented for quality assurance and worker evaluation. 
🛠 Technical Implementation & FeaturesReal-time Communication: Built-in chat system for direct negotiation between client and worker.  Voice Integration: Ability for clients to call workers directly through the app.  Location Services: Integrated GPS/Maps functionality to help clients find local workers.  Service Payments: Built-in payment flow logic. Note: Integrated with PayDunya (Prototype stage).  Notifications: Push notifications to keep users updated on order status and price offers.  
💻 Technical Toolkit   Mobile Development: Flutter for a smooth, high-end user experience.  Backend: Node.js and Express.js for handling the booking logic and user authentication.  Languages: Expert use of Java, C++, and SQL.  Database: SQL and Database Management for structured data handling (Users, Bookings, Reviews).  🎓 Academic BackgroundDeveloped by Marwa Yhayaoui, a full-stack developer and student at ESI SBA.
